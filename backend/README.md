# CoderRooms Backend

Realtime API, Socket.IO room updates, and local code judging for JavaScript, Python, and Java.

## Run

```bash
npm install
npm run dev
```

The API starts on `http://localhost:4000`.

No paid API key is required. Code is executed locally with a 3 second timeout, which is useful for development. For a public deployment, run the judge inside locked-down containers or use a hosted judge service.
